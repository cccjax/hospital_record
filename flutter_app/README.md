# 住院管理系统 Flutter 复刻（进行中）

本目录是基于现有 Web 原型迁移的 Flutter 版本，目标是实现同等的多端离线能力（iOS/Android）。

## 当前已完成

- 三大底部 Tab：`首页` / `模板` / `我的`
- 首页病人列表：搜索、在院筛选、新增/编辑/删除、点击进入病人明细
- 病人明细：基础档案编辑、入院记录列表、入院新增与限制（已有在院记录时禁止重复新增）
- 入院详情：
  - 入院详情展示与编辑
  - 日常记录新增/编辑/删除/查看详情
  - 住院测评新增/编辑/删除/查看只读详情
  - 影像资料拍照/相册上传、缩略图预览、删除
  - 测评分数区间进度条展示
- 模板模块：
  - 病种搜索
  - 病种展开/收起版本列表
  - 病种、版本、测评项、分级区间的增删改
- 我的模块：
  - 数据迁移（导入/导出 JSON）
  - 密码保护（开启/修改/关闭）
  - 字段配置（新增/编辑/删除、显示隐藏、排序模式）
- 应用启动密码校验页（启用密码后生效）

## 目录结构

- `lib/src/models`：数据模型与默认数据
- `lib/src/state`：全局状态与业务逻辑
- `lib/src/pages`：页面层
- `lib/src/widgets`：通用组件

## 运行方式

在 Flutter 环境可用后执行：

```bash
flutter pub get
flutter run
```
