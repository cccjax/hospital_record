# 住院管理系统 Flutter 版

本目录为住院管理系统的 Flutter 实现（离线优先），用于复刻并持续优化 Web 原型的业务逻辑与交互体验。

## 当前能力
- 底部三大模块：`首页` / `模板` / `我的`
- 病人管理：检索、在院筛选、增删改查、详情联动
- 入院管理：入院记录增删改查、在院约束校验
- 日常记录：增删改查 + 详情
- 住院测评：按模板/版本完成测评，支持新增、编辑、删除、只读查看
- 影像资料：拍照上传、相册上传、预览、删除
- 字段配置：动态新增/编辑字段、可见性控制、排序
- 数据迁移：文件导出/导入（JSON）
  - 导入前校验：编码、JSON 格式、结构与类型、可解析性、空数据拦截
  - 导入确认：覆盖提醒 + 数据摘要确认
- 安全能力：应用访问密码（启用/修改/关闭）

## 目录结构
- `lib/src/models`：数据模型与默认数据
- `lib/src/state`：状态管理与业务逻辑
- `lib/src/pages`：页面层
- `lib/src/widgets`：通用组件
- `lib/src/theme`：全局主题

## 开发命令
在 `flutter_app` 目录执行：

```bash
flutter pub get
flutter analyze
flutter test
flutter run -d emulator-5554
```
